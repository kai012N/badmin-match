# 羽球排場管理系統 — 產品迭代紀錄

> 一個專為羽球球聚設計的智慧排場工具，從概念到產品的完整開發歷程。

---

## 產品簡介

**羽球排場管理系統**解決的是每次球聚最麻煩的問題：「這一局誰上？怎麼分隊？誰等太久了？」

傳統做法靠人工輪流，容易出現有人連打、有人一直等、等級差距太大等問題。這個系統用演算法自動排場，兼顧公平性與競技平衡。

### 核心功能
- **智慧配對**：依等待時間、等級差距自動推薦最佳 4 人組合
- **即時指派**：支援拖曳、點選兩種操作方式，手機友善
- **連打保護**：設定連打上限，強制輪換讓每個人都能休息
- **歷史追蹤**：記錄隊友／對手配對歷史，避免重複組合
- **比分記錄**：每局結束輸入比分，驅動動態評分系統自動調整
- **動態 Rating**：Elo + Rating Deviation，配對越打越準確
